All consoles can join custom Bedrock Edition servers with some minor setup. These methods make use of a program called 'BedrockConnect', which allows you to join custom servers.

Another service like BedrockConnect is Geyserconnect. It's like BedrockConnect but to also join Java Edition servers utulizing GeyserMC!

Playstation 4

To set it up on PS4, it's a bit different, so heres steps.

Step 1 Go to your PS4 Home screen.
Step 2 Go to Settings.
Step 3 Go to Network.
Step 4 Select Set Up Internet connection.
Step 5 If you are using wired internet, select "Use LAN Cable", otherwise choose "Use Wi-Fi".
Step 6 Select the Custom network creation mode.
Step 7 Select Automatic IP Address.
Step 8 For DHCP Host Name, make sure you select Do Not Specify.
Step 9 Under DNS Settings, select Manual.
Step 10 Enter the BedrockConnect IP for the preferred Primary DNS (There are many options for BedrockConnect ServersFind the one that best suits you.) and something like Google or Cloudfare's IP for the Secondary DNS (8.8.8.8 or 1.1.1.1).

The IP is play.gamersclub.gq
